# language-ldpl

Adds syntax highlighting, code folding and snippets to [LDPL](https://www.ldpl-lang.org/) files in Atom.

This package uses the [tree-sitter-ldpl](https://www.npmjs.com/package/tree-sitter-ldpl) grammar.
